mPythonX IDE
====================

软件安装
-----------

目前有两个下载地址：

http://static.steamaker.cn/files/mPythonXSetup0.0.9.exe
(Win 7 / 8 / 10、MacOS)

http://static.steamaker.cn/files/mPythonXSetup0.0.9_XP.exe
(Win XP)


.. Hint::
  
  安装过程有可能被杀毒软件误报病毒，需要选择“允许程序所有操作”。


安装软件的最后一步，会自动安装CP210x的驱动，如果先前安装过则可以忽略。
